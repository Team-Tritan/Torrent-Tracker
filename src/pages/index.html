What does a torrent tracker do?

A torrent tracker is a lightweight coordination service used by the Torrent protocol. When a user opens a torrent or magnet link, their Torrent client contacts the tracker to obtain a list of peers currently sharing the same content.

The tracker does not host, cache, or distribute any files. It only facilitates communication between peers participating in a swarm.

How peer discovery works

  1. A user opens a .torrent file or magnet link.
  2. The torrent client announces itself to the tracker using the configured announce URL.
  3. The tracker returns a list of active peers participating in the swarm.
  4. The torrent client exchanges file pieces directly with peers over a peer-to-peer connection.

All content transfers occur directly between users — never through the tracker itself.

Using this tracker

Add the tracker announce URL to your torrent configuration or magnet link: 

tracker.tritan.gg:8080/announce

Supported by most modern torrent clients including:

  • qBittorrent
  • Transmission
  • Deluge
  • rTorrent
  • aria2

Example magnet link

magnet:?xt=urn:btih:HASHHERE&tr=udp://tracker.tritan.gg:8080/announce
